Sedna Length-frequency Import Form:

  • the file must be in csv format
  • This function is designed to import a csv export of herring sample length frequency data collected in Sedna
  • the order of the columns in the file is NOT important
  • IT IS ASSUMED THAT ALL THE CORRESPONDING SAMPLES HAVE BEEN ADDED!!
  • the file must have the follow format:
sample_uuid length_bin count
This is a unique universal identifier assigned to a sample from sedna. This will be stored as the old_id under the Sample object for historical reference This is the length frequency bin This is the count of the number of fish in that bin for the given sample
